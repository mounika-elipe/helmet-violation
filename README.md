This project detects whether a two-wheeler rider is wearing a helmet using the YOLOv8 object detection model. It identifies helmet violations in real-time from video input to support traffic safety systems.

## Features
- Helmet and non-helmet detection
- Real-time video processing
- YOLOv8-based object detection
- Supports image and video inputs

## Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy
- Pandas
- Jupyter Notebook

## How It Works
1. Input video/image is given
2. YOLOv8 detects helmet presence
3. Rider is classified as helmet / no-helmet
4. Output is displayed with bounding boxes
